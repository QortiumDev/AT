# Authors and attribution

This repository preserves the complete Git history of the Automated
Transactions implementation now maintained by QortiumDev. The commit history
is the authoritative record of individual authorship.

The principal project lineages and contributors are:

- **CIYAM Developers / Ian Knowles** — original Automated Transactions work,
  specification, documentation, and project stewardship.
- **catbref** — Java reimplementation and subsequent virtual-machine,
  toolchain, and test development through the 1.4.x lineage.
- **IceBurst** — dependency, Maven, Javadoc, and JitPack modernization through
  version 1.4.3.
- **bold** — an inherited test-output correction.
- **QortiumDev contributors** — continued maintenance and Qortium-specific
  correctness work.

The repository lineage is:

`ciyam/AT` -> `catbref/AT` -> `IceBurst/AT` -> `QortiumDev/AT`

QortiumDev maintains this as an independent GitHub repository rather than as
a GitHub-network fork. That organizational independence does not alter or
replace the original commit authorship.
